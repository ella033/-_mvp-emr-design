# examples

이 디렉터리는 기존 제품에서 사용하던 서식/인쇄 기능 소스를 참고용으로 보관하기 위한 공간입니다.\
현재 프런트엔드 애플리케이션에서는 사용되지 않으며, 빌드나 배포에 포함되지 않습니다.

## 구성

- `print-forms/`
  - `Html/`: 이전 WinForms 기반 제품에서 사용하던 정적 HTML, CSS, 이미지 예제
  - `Maker/`, `Common/`: 해당 제품의 서버/클라이언트 렌더러에서 쓰이던 C# 헬퍼 코드

## 사용 시 주의 사항

- 코드나 자산을 복사하여 사용할 때는 최신 문서 모듈 구조 및 디자인 시스템과 호환되는지 반드시 확인하세요.
- 샘플은 레거시 구현을 보여주기 위한 것이므로, 직접 수정하거나 재배포하지 않는 것을 권장합니다.
